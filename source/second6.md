# 错误码汇总

| HTTP 状态码 | CODE              | 说明             |
|----------|-------------------|----------------|
| 500      | INTERNAL_ERROR    | 内部服务错误         |
| 404      | NOT_FOUND         | 访问信息不存在或暂时查询不到 |
| 403      | FORBIDDEN         | 无访问权限          |
| 400      | BAD_REQUEST       | 参数错误           |
| 400      | DUPLICATE_REQUEST | 重复请求           |
| 400      | STATUS_ERROR      | 状态异常           |
